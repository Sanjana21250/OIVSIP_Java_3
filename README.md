# ATM Interface (Java Project)

## Objective
The objective of this project is to simulate a basic ATM (Automated Teller Machine) system using Java. It allows users to perform banking operations like withdrawal, deposit, transfer, and viewing transaction history.

## Tools & Technologies Used
1. Java (Core Java)
2. VS Code / IntelliJ IDEA
3. Git & GitHub

## Features
* User authentication (User ID & PIN)
* Withdraw money
* Deposit money
* Transfer funds
* Transaction history tracking

## Steps Performed
1. Designed the ATM class with basic functionalities
2. Implemented user authentication system
3. Created menu-driven interface
4. Added transaction history using ArrayList
5. Handled balance updates for each operation
6. Tested all functionalities

## How to Run
1. Compile the program:
   javac ATMFunctionality.java
   ATMInterface.java
3. Run the program:
   java ATMInterface
   
## Outcome
This project demonstrates the use of:
* Object-Oriented Programming (OOP)
* Conditional statements
* Loops
* Collections (ArrayList)
It provides a simple and effective simulation of ATM operations.

## Conclusion
The ATM Interface project helped in understanding real-world banking logic and strengthened Java programming fundamentals.
